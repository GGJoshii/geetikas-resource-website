ACCESSIBILITY

Introduction

The principle that everyone should be able to use the web equally, regardless of any disability, is as important today as it was twenty years ago. People who have limited use of their arms, people who cannot hear or see well and people who process information differently, should all be able to access and use web sites and apps. Web technologies have many built in features for accessibility, and we as web developers should know them so that we do not exclude any of our users. There is no precise starting point of the history of web accessibility.

Semantic Elements

I provide the user a good way to navigate and interact with my site. Semantics is about using the correct element in HTML. There are approximately 110 elements in HTML 5.

Color Contrast

I measured the contrast between text or graphics against the background color. This is called contrast ratio. A white text on a white background has a contrast ratio of 1. This is impossible to perceive. Black text on a white background has a contrast ratio of 21.

Alt Text

The alternative text for an image is for users that for some reason cannot view it. The reason can be a slow connection, an error with the image file, or if the user uses a screen reader. The value of the alt attribute describes the image, or even better the intention of the image.
